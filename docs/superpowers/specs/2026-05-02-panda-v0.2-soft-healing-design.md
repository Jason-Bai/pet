# Panda v0.2 Soft Healing Design

## Goal

Panda v0.2 refreshes the existing Codex custom pet as a softer, more healing companion. The pet should feel like a quiet panda sitting beside the user while they code: warm, gentle, a little sleepy, and emotionally supportive.

This version updates the visual animation sheet only. It does not add new runtime behavior.

## Non-Goals

- Do not add hover, click, petting, or custom pointer APIs. Current Codex custom pet manifests expose only `id`, `displayName`, `description`, and `spritesheetPath`.
- Do not change the package layout. The install path remains `~/.codex/pets/panda`.
- Do not add accessories, text, UI symbols, speech bubbles, or scenery.
- Do not make the panda loud, meme-like, overly bouncy, or visually busy.

## Visual Direction

- Keep the v0.1 panda identity: round white head, black ears, black eye patches, tiny paws, short legs, soft blush, and thick pixel-adjacent outline.
- Make expressions gentle instead of exaggerated.
- Use small body movement, breathing, blinking, head tilts, and paw poses to create comfort.
- Keep effects attached to the body if used at all, so transparent extraction stays reliable.
- Preserve the Codex digital pet style: compact chibi silhouette, simple palette, flat shading, readable poses, and no polished illustration rendering.

## State Plan

| State | v0.2 Behavior |
| --- | --- |
| `idle` | Slow breathing, soft blink, tiny cheek warmth, calm seated posture. |
| `running-right` | Short-legged toddle to the right, still round and soft rather than frantic. |
| `running-left` | Mirrored from `running-right` if the gait and markings remain safe when flipped. |
| `waving` | Gentle paw raise, like a quiet hello; no wave marks or detached effects. |
| `jumping` | Small happy hop with soft landing posture; no dust, shadow, or impact marks. |
| `failed` | Mild sad expression, tiny attached tear or lowered paws, healing rather than dramatic. |
| `waiting` | Patient seated pose, paws resting near belly, occasional head tilt or sleepy blink. |
| `running` | Soft in-place trot, useful for generic activity without high energy. |
| `review` | Attentive lean and calm focus, like quietly checking the work with the user. |
